Prerequisites
--------------
Make sure you have python, pygame and numpy installed.
This program was tested with Python 2.7.


How to play
-----------

    $ python missile_defence.py

Use the mouse to aim and fire.


Keys:  q   quit
       r   reset
       a   auto aiming and firing (e.g. for use as a screensaver)
